## Installation

```zsh
$ npm install --global iapid
```

## Usage

```zsh
$ iapid <command> <subcommand>
```

### Commands

- `new`
- `delete`
- `generate`
- `list`
- `set`
- `show`

### Subcommand

- `template`
- `project`

## Examples

```zsh
$ iapid new project
$ iapid new template
$ iapid set
$ iapid generate --select --display
```
